# Exercises Create a new repository and name it CSS Exercises. - [ ] Exercise 1: Basic Box Model - Create a simple webpage with three boxes. Apply different background colors, borders, and margins to each box. Experiment with padding to see how it affects the content within the boxes. - [ ] Exercise 2: Centering Elements - Create a container and center its content both horizontally and vertically using margin and padding. - [ ] Exercise 3: Floating Elements - Use the float property to create a layout where three boxes are side by side. Experiment with clearing floats to avoid layout issues. - [ ] Exercise 4: Responsive Design with Media Queries - Make your webpage responsive by using media queries. Adjust the layout, font sizes, or any other styles to provide a good user experience on both desktop and mobile devices. - [ ] Exercise 5: Flexbox Layout - Create a navigation bar using Flexbox. Make sure the items are evenly spaced and aligned horizontally. Add responsiveness using Flexbox properties. - [ ] Exercise 6: Grid Layout - Build a grid-based layout with CSS Grid. Create a section with multiple items, and use grid properties to define the layout. - [ ] Exercise 7: Complex Layout - Combine Flexbox and CSS Grid to create a more complex layout. For example, have a header, sidebar, main content area, and a footer. Experiment with different arrangements and sizes. - [ ] Exercise 8: Card Layout - Design a card layout commonly used for displaying content like articles or product cards. Use the box model, and consider adding hover effects or transitions for a polished look. - [ ] Exercise 9: Nesting Elements - Create a structure where boxes are nested inside each other. Apply different styles to parent and child elements to see how they interact.
